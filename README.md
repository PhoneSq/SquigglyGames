# SquigglyGames
C++ games (Pong, Snake, Breakout) built from scratch using Win32 API

A retro-style Windows game suite built with **Win32 UI** in C++.  
Dive into a green-on-black arcade world with **Pong**, **Breakout**, and **Snake**, all handcrafted with classic vibes.

## Games Included

### Pong
- Play against a simple **AI** or a **friend** in local multiplayer.  
- Smooth paddle controls and fast-paced ball physics.

### Breakout
- Classic brick-breaking action with **particle effects**.  
- Keeps track of your **score**.

### Snake
- No grid restriction—move freely across the screen!  
- Grow longer, dodge walls, and aim for the **high score**.

## Features
- Retro **green-on-black** aesthetic for maximum nostalgia.  
- Simple **Win32 UI**, no external engines required.  
- Scores and particle effects for extra arcade feel.  
- Lightweight and fun—runs on standard Windows setups.
